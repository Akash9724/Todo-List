A simple and interactive To-Do List application built using React.js.
This app helps users add, manage, and organize daily tasks efficiently.
The app uses Local Storage to persist tasks, so your to-dos stay saved even after refreshing the page.

### Prerequisites
- Node.js (v16 or above)
- npm

### Setup
- npm install
- npm run dev
